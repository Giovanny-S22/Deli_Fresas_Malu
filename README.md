# 🍓 Deli Fresas Malú — Menú Digital QR

> Menú digital responsive para acceder desde código QR. Los clientes escanean en la mesa y ven productos, precios y contacto desde su celular.

---

## Descripción

Landing page desarrollada para **Deli Fresas Malú**, una tienda de fresas con crema y postres ubicada en Colombia. Reemplaza el menú físico por una página web accesible mediante QR, optimizada para móvil.

Proyecto construido como práctica de desarrollo web frontend.

---

## Stack tecnológico

| Tecnología | Uso |
|---|---|
| HTML5 | Estructura de la página |
| CSS3 | Estilos y diseño responsive |
| JavaScript | Interacciones y navegación |
| GitHub Pages | Hosting gratuito |

---

## Características

- Diseño responsive (mobile-first)
- Categorías de productos con foto y precio
- Sección de contacto y redes sociales
- Acceso directo por código QR
